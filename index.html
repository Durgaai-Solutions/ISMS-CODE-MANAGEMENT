HOW IT WORKS

Get Connected In 3 Steps

One School Code — Two Apps. Your Code Links Both The Parent & Teacher Apps To Your School's Private Backend.

1

School Onboarding

Durgaai Solutions Sets Up Your School's ISMS Backend And Assigns A Unique School Code. The Code Is Registered Here So The App Can Resolve It Instantly.

2

Parent Credentials

The School Admin Generates A PARENT-XXXXX Username And Password For Each Student Directly From The ERP Dashboard.

3

Login In The App

Parents Open The ISMS Parents Portal App, Enter Their School Code + Username + Password, And Get Instant Access To Fees, Attendance, Results & More.

1

School Onboarding

2

Teacher Credentials

3

Login In The Teacher App

Teachers Open The ISMS Teacher Portal App, Enter Their School Code + Username + Password, And Access Timetables, Attendance, Mark Entry & Notices.
