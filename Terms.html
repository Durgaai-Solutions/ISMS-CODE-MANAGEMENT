01 User Agreement & Eligibility

By Accessing The ISMS Core Ecosystem (Admin Portal, Teacher App, Parents App) Provided By Durgaai Solutions, Your Educational Institution And Its Authorized Users Agree Strictly To Use The Software Per The Contracted Deployment Terms.

Access To ISMS Core Is Granted Only To:

  • School Administrators And Principals: Via The ISMS ERP Admin Portal — Requires Valid School Code And Admin Credentials Issued By Durgaai Solutions.
  • Teaching And Non-Teaching Staff: Via The ISMS Staff Application — Credentials Provisioned By The School's Admin Account.
  • Parents And Guardians: Via The ISMS Parents App — Account Linked To A Registered Student Profile Created By The School.
  • Authorized IT Personnel: For System Maintenance And Integration Purposes Under A Documented Technical Support Agreement.
